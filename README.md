# 📝 React Notes App

A simple Notes App built with React to understand **state management, data flow, and component architecture**.

This project is a rebuild of my earlier vanilla JavaScript notes app, keeping the same UI to focus on how React handles state and updates.

---

## 🚀 Features

- Add notes  
- Edit notes  
- Delete notes  
- Search notes  
- Pin / Unpin notes (pinned notes appear first)

---

## 🧠 What I focused on

- Lifting state up  
- One-way data flow  
- Controlled components  
- Immutable updates using `map` and `filter`  
- Conditional rendering  

---

## 🛠 Tech Stack

- React (Vite)
- JavaScript (ES6)

---

git clone https://github.com/den766/react-notes-app
